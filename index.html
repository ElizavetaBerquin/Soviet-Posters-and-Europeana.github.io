Click 'Load the posters!' button once to display the first 500 posters available on the Europeana APIs.

You will see all the posters as a single grid of images sorted by colors extracted from metadata. This method allows to explore large numbers of graphic objects, while still being able to see each image separately. When all the images are displayed simultaneously it makes it easier to compare them and observe patterns, differences and similarities in composition or color.

Please note that loading the posters may take a few minutes.

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
